# 기능 설명

## **스도쿠 채점기능**
완성시킨 스도쿠가 올바른 스도쿠인지 채점합니다.

## **가로ㆍ세로줄 및 3x3블럭 하이라이트**
선택한 칸의 가로ㆍ세로 및 3x3블럭에 하이라이트를 주어, 스도쿠를 완성하는 데에 편리합니다.
